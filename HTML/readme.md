1. Consulte y explique que significa a nivel de uso del internet: 
a. Servidor DNS

Un Servidor DNS (Domain Name System) es como la “agenda telefónica” de Internet.
Traduce nombres fáciles de recordar (como google.com) en direcciones IP que las computadoras entienden.

 Ejemplo:
Escribes youtube.com → el DNS lo convierte en algo como 142.250.190.14

b. IP (Dirección IP)

Es un número único que identifica a cada dispositivo conectado a Internet.

Ejemplo:
192.168.1.1

Sirve para que los datos sepan a dónde ir y de dónde vienen.

c. Protocolo

Un protocolo es un conjunto de reglas que permiten la comunicación entre dispositivos.

 Ejemplos comunes:

HTTP → para páginas web
HTTPS → versión segura
FTP → transferencia de archivos

d. Nombre de dominio

Es el nombre fácil de recordar de un sitio web.

 Ejemplo:
facebook.com

Es una forma amigable de representar una IP.

e. Directorio y nombre de archivo

Es la ruta específica dentro de un sitio web donde se encuentra un recurso.

 Ejemplo:
www.ejemplo.com/carpeta/pagina.html

carpeta → directorio
pagina.html → archivo

Sirve para ubicar contenido específico dentro de un sitio.

2. Mencionamos dos roles importantes en el desarrollo de un producto web, pero hay 
otros roles involucrados, consulte y defina qué hace cada uno de los siguientes 
roles: 

a. Diseñador UI/UX

Se encarga de la experiencia del usuario.

UI (User Interface) → diseño visual (botones, colores, tipografía)
UX (User Experience) → cómo se siente usar el producto

 Objetivo: que el producto sea fácil, intuitivo y agradable.

b. Cloud Manager / Ingeniero de infraestructura en la nube

Administra los servidores y servicios en la nube.

Configura hosting (AWS, Azure, etc.)
Escalabilidad (que soporte muchos usuarios)
Seguridad y disponibilidad

 Es quien asegura que la web funcione siempre y sin caídas.

c. Administrador de base de datos (DBA)

Gestiona la información del sistema.

Diseña bases de datos
Optimiza consultas
Protege los datos

 Se encarga de que los datos estén seguros, organizados y accesibles.

d. Project Manager / Director del Proyecto

Coordina todo el proyecto.

Define tiempos y objetivos
Organiza el equipo
Supervisa el progreso

Es quien garantiza que el producto se entregue a tiempo y correctamente.


3. ¿Qué significa que un lenguaje sea turing completo? 

Un lenguaje es Turing completo si puede resolver cualquier problema computacional (si se le da suficiente tiempo y memoria).

 En otras palabras:
Puede ejecutar cualquier algoritmo.

Ejemplos:

Python
JavaScript
C++

4. ¿Por qué CSS y HTML no son lenguajes de programación? 

Porque no tienen lógica de programación completa.

No pueden hacer decisiones complejas (if, while, etc.)
No manejan lógica ni algoritmos

 HTML → estructura (contenido)
 CSS → diseño (apariencia)

No “piensan”, solo describen.
